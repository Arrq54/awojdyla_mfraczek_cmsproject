# awojdyla_mfraczek_cmsproject <br>
Projekt przygotowujemy w zespołach dwuosobowych i jednym trzyosobowy. Postępy prac będą sprawdzane na kolejnych lekcjach aż do dnia oddania projektu do oceny. Do całości projektu należy przygotować dokumentację, a wyniki prac należy zamieścić na serwerze Git dostępnym w sali 514i.  Projekt będzie oceniany przez dwóch prowadzących w ramach przedmiotów Aplikacje internetowe i Aplikacje desktopowe. Do uzyskania jest w sumie 30pkt.:  Aplikacje internetowe (frontend) - 10pkt. Aplikacje internetowe (backend) - 10pkt. Aplikacje desktopowe - 10pkt.<br><br>
